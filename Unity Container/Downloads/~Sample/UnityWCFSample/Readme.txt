This sample includes two hosts and a single client.
Host #1: Self-hosted WCF Service (CalculatorServiceSelfHost project)
Host #2: IISWAS Service (CalculatorServiceIISWASHost project)

You must run Visual Studio with elevated permissions if you plan to run the Self-hosted WCF Service.
You must change the client configuration to connect to one or other of the services: see App.config for more details.
